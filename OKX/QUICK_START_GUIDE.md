## 🎓 策略原理

### 入场信号
- **做多**: 短期均线(MA5)向上突破长期均线(MA15)，且价格高于MA5
- **做空**: 短期均线向下突破长期均线，且价格低于MA5

### 出场条件
- **止盈**: 盈利达到4-4.5%
- **止损**: 亏损达到1.5-2%
- **市价单**: 快速成交，减少滑点

## 💰 实战技巧

1. **选择时段**:
   - 最佳: UTC 12:00-16:00 (亚洲盘)
   - 次佳: UTC 20:00-00:00 (欧美盘)
   - 避免: 低波动时段

2. **趋势过滤**:
   - 只在趋势明显时交易
   - 震荡市减少交易频率

3. **资金管理**:
   - 单次风险不超过5%账户
   - 连亏3次后，减半仓位
   - 连赢5次后，提取部分利润

4. **心理控制**:
   - 不要追涨杀跌
   - 不要在亏损后报复性交易
   - 达到日盈利目标后休息

## 📞 问题排查

**Q: 程序提示余额不足？**
A: 检查杠杆是否设置成功，或降低开仓数量

**Q: 策略不开仓？**
A: 需要等待20个价格数据点才会开始交易，耐心等待

**Q: 频繁止损？**
A: 可能市场震荡，考虑：
   1. 增加MA周期长度
   2. 减少交易频率
   3. 暂停交易等待趋势

**Q: 如何提高胜率？**
A:
   1. 只在明显趋势中交易
   2. 避免震荡市
   3. 优化止盈止损比例
   4. 增加均线周期（减少假信号）
