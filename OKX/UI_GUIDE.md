## 📊 UI界面功能

### 左侧面板

#### 📈 市场行情
- **当前价格**: 实时PEPE价格
- **24h涨跌**: 当日波动率
- **24h成交**: 成交量

#### 💰 账户信息
- **账户余额**: 总权益
- **可用余额**: 可开仓金额
- **保证金占用**: 已用保证金比例

#### 📊 当前持仓
- 持仓方向（做多/做空）
- 持仓数量
- 开仓价格
- 浮动盈亏

### 右侧面板

#### 🎮 控制按钮
- **▶ 启动交易** - 启动机器人
- **⏸ 停止交易** - 停止机器人
- **🔄 刷新数据** - 手动刷新
- **⚙ 配置** - 查看策略配置

#### 📝 实时日志
- 显示交易信号
- 开平仓记录
- 系统状态

## 🎯 使用步骤

### 1. 启动UI
```bash
python trading_ui.py
```

### 2. 检查数据
- 查看市场行情是否正常显示
- 确认账户余额正确
- 检查是否有持仓

### 3. 启动交易
- 点击 **"▶ 启动交易"** 按钮
- 观察日志输出
- 等待交易信号

### 4. 监控交易
- 实时查看持仓变化
- 关注浮动盈亏
- 注意风险提示

### 5. 停止交易
- 点击 **"⏸ 停止交易"** 按钮
- 机器人会安全退出

## 🛠️ 故障排除

### UI无法启动
```bash
# 检查Python版本
python --version  # 需要Python 3.7+

# 安装依赖
pip install tkinter  # 通常Python自带
```

### 无法连接OKX
1. 检查代理是否启动（127.0.0.1:7897）
2. 测试代理: `python test_proxy.py`
3. 检查API密钥是否正确

### 机器人不交易
1. 查看实时日志，确认是否有信号
2. 检查账户余额是否充足（≥$5）
3. 确认信号强度是否满足条件
4. 等待5-10分钟收集数据
